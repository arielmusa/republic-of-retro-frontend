# Republic of Retro

Republic of Retro is an e-commerce application where users can browse products,use filters, view details and do cart and checkout actions.

## Project Structure
```
src/
  ├── assets/
  ├── components/
  ├── context/
  ├── Pages/
  ├── services/
  ├── types/
  ├── App.jsx
  └── main.jsx
```
## Run Locally

Clone the project

```bash
  git clone https://github.com/arielmusa/republic-of-retro-frontend.git
```

Go to the project directory

```bash
  cd republic-of-retro-frontend
```

Run the project

```bash
npm install
npm run dev
```

Application will run at:
http://localhost:5173
